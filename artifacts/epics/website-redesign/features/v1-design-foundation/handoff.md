# Feature Handoff — v1-design-foundation

## Intent
Establish the project-level visual and brand foundation required before any V1 page redesign / implementation work proceeds.

## Scope
- Create `design/DESIGN_SYSTEM.md`
- Create `design/BRAND_GUIDELINES.md`
- Align the visual foundation with the approved V1 positioning and existing Light Premium Bento direction

## Inputs
- `artifacts/project/BRIEF.md`
- `artifacts/project/REQUIREMENTS.md`
- `artifacts/project/ARCHITECTURE.md`
- `artifacts/project/PROJECT_MEMORY.md`
- `artifacts/project/GOTCHAS.md`
- `SITE_V2_DESIGN_SYSTEM.md`
- `REDESIGN-PROPOSAL.md`
- Current implemented pages under `app/[lang]/`

## Non-goals
- Final page-by-page redesign
- Code changes
- Blog / news planning

## Done Criteria
- `design/DESIGN_SYSTEM.md` exists and defines colors, typography, spacing, component styles, states, and usage rules
- `design/BRAND_GUIDELINES.md` exists and defines brand expression, icon / illustration direction, and do/don't guidance
- The output reflects the current V1 business positioning:
  - AI社会実装のパートナー
  - 「AI」を、現場の「即戦力」へ
  - 「属人」のノウハウを、組織の「資産」に
  - Web3.0テクノロジーを、ビジネスへ
