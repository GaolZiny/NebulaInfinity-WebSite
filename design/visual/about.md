# Visual Spec — About

## 1. Page role
- Primary goal: explain who Nebula Infinity is and why the company is a credible implementation partner.
- Audience: prospects validating fit, trust, and philosophy after understanding services.
- Primary action: move to Contact.
- Secondary action: move to Services.

## 2. Header / navigation flow
- Header: global shell.
- Transitions:
  - Service line cards -> respective service detail pages
  - CTA -> Contact / Services
- Back destination: Home via global nav/logo.
- Unauthenticated flow: fully open.

## 3. Hi-Fi ASCII layout

### Desktop
```text
Header
Hero: About eyebrow + positioning line + subtitle
Positioning pillars: 4 cards using approved message anchors
What we do: 3 service-line cards
How we work: 4 principle cards
Company snapshot: location / domain / focus / representative products
CTA band
```

### Mobile
```text
Header
Hero
Positioning pillars x4
Service cards x3
Principle cards x4
Company snapshot stacked rows
CTA band
```

## 5. Global state checklist
- Service cards: default / hover / focus-visible.
- CTA buttons: default / hover / pressed / focus-visible / disabled.

## 6. Responsive summary

- Keep proof references selective; do not turn the page into a second Projects index.
